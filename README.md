# AI Student Chatbot

An offline AI-powered chatbot developed as a college Artificial Intelligence minor project. The chatbot answers frequently asked student queries related to admissions, courses, fees, placements, hostel, scholarships, library, college timings, and other campus information.

The application is built using Python, Flask, and Scikit-learn. It uses TF-IDF Vectorization and Cosine Similarity to find the most relevant answer from a dataset of over 100 FAQs without relying on external APIs or internet connectivity.

## Features

- Offline chatbot (No paid APIs required)
- 100+ Frequently Asked Questions
- TF-IDF based text vectorization
- Cosine Similarity matching
- Responsive ChatGPT-inspired interface
- Flask backend
- CSV-based FAQ dataset
- Fast and lightweight
